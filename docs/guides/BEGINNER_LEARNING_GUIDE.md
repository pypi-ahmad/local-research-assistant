# Beginner Learning Guide

## What this project teaches
- How production RAG systems are structured.
- How ingestion transforms raw files into searchable knowledge.
- How citations reduce hallucination risk.
- How to combine relational, vector, graph, and object storage.

## Suggested learning path
1. Start API and open `/docs`.
2. Create user and notebook.
3. Upload one PDF and one CSV.
4. Run search and inspect retrieval hits.
5. Ask chat question and review citations.
6. Generate flashcards/quizzes.
7. Inspect graph query output.

## Key engineering ideas
- Separation of concerns across services.
- Immutable source artifacts + derived indexes.
- Async workers for long-running pipeline steps.
- Monitoring and backup as first-class production features.
